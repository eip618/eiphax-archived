A guide to using FTP for the 3DS

This guide will teach you how to use FTP or file transfer protocol. The benefits of FTP being that you can move files wirelessly between your pc and 3DS. FTP is much safer than microsd management and is not all too difficult to set up. Guide written by Krieg#1473 on Discord.

Requirements: A FTP client on your 3DS, and an FTP client on your computer.

For PC I use and recommend winscp. For 3DS we are going to use FTPD. You can scan the QR code on the page by going to FBI then remote install then scan qr code. If you cannot use QR codes, all you need is the .CIA.

Setting up FTPD.

1. Install FTPD on your 3DS.
2. Open it.
3. Look at the top screen, right next to ftpd v2.3.0. What is highlighted in blue is your IP address, what is highlighted in red is your port.
4. Now, keep FTPD open.

Setting up WinSCP

1. Run the WinSCP installer and wait till it is done downloading.

2. You should be greeted to this once opened.

3. Set SFTP to FTP and enter your host name and port number.

4. Make sure "Anonymous login" is checked.

5. Your setup should now look like this.

6. Press connect.

6.5 If you do not want to input your IP address and port, you can save the config by pressing the save button.

7. If successful, you will be able to move files wirelessly between your 3DS and computer!
